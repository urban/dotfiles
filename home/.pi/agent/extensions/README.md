# Pi Extensions

Special thanks to Armin Ronacher (@mitsuhiko) for creating and open-sourcing these extensions. Original repository: https://github.com/mitsuhiko/agent-stuff. 

- `answer.ts` - Structured Q&A for complex queries
- `context.ts` - Context breakdown dashboard
- `files.ts` - Unified file browser with git integration
- `loop.ts` - Iterative execution loop
- `notify.ts` - Desktop notifications via OSC 777
- `review.ts` - Code review system
